### 🎬 تسلسل الحركات (Motion Sequence)

بعد السماح بالكاميرا ستشاهد:

```
┌─────────────────────────────────┐
│     LIVE CAMERA STREAM          │
│   [Your Face Here]              │
│                                 │
│   Instruction: لف رأسك لليمين   │
│   Step 1 of 6                   │
└─────────────────────────────────┘
```

#### الحركة 1: لف الرأس لليمين
```
1. ضع وجهك أمام الكاميرا
2. نظر مباشرة للكاميرا
3. لف رأسك ببطء لليمين (45-60 درجة)
4. ستشاهد شريط تقدم يزداد (0-100%)
5. عندما يصل 100% سيظهر "✓ Turn Head Right detected!"
```

**✅ متوقع:**
- وجهك معروف تلقائياً (رسالة خضراء في الأسفل)
- التعليمات واضحة وباللغتين
- شريط التقدم يتحرك مع حركتك

#### الحركة 2: لف الرأس لليسار
```
1. عد إلى الأمام
2. لف رأسك ببطء لليسار (45-60 درجة)
3. شريط التقدم يزداد
4. عندما يصل 100% → "✓ Turn Head Left detected!"
```

#### الحركة 3: ارمش بعينيك
```
1. ضع وجهك أمام الكاميرا
2. انظر مباشرة
3. أغمض عينيك بسرعة (كأنك تُرمّش)
4. افتح عينيك
5. كرر مرتين أو ثلاث
6. سيظهر "✓ Blink Eyes detected!"
```

#### الحركة 4: افتح فمك
```
1. انظر مباشرة
2. افتح فمك بشكل واضح (كأنك تقول "آآآ")
3. ثبّت الفم مفتوح لثانية واحدة
4. أغلق الفم
5. سيظهر "✓ Open Mouth detected!"
```

#### الحركة 5: قرّب وجهك
```
1. ابدأ بمسافة 30-40cm من الكاميرا
2. اقترب ببطء من الكاميرا
3. اقترب حتى 15-20cm
4. شريط التقدم يزداد مع اقترابك
5. سيظهر "✓ Move Close detected!"
```

#### الحركة 6: ابعد وجهك
```
1. ابدأ من موقع قريب
2. ابتعد ببطء عن الكاميرا
3. ابتعد حتى 40-50cm
4. شريط التقدم يزداد مع ابتعادك
5. سيظهر "✓ Move Away detected!"
```

**🎉 بعد إكمال جميع الحركات:**
```
✓ تظهر رسالة "Verification Complete!"
✓ صورة تُلتقط تلقائياً
✓ شاشة تنتقل للخطوة التالية
✓ رسالة "Biometric liveness verification complete!"
```

## 🆘 استكشاف الأخطاء الشائعة

### المشكلة: "Face not detected"
```
❌ الخطأ: وجهك لا يُكتشف

✅ الحلول:
1. تأكد من الإضاءة الجيدة
2. ضع وجهك مباشرة أمام الكاميرا
3. ابتعد قليلاً (25-40cm)
4. تأكد من وضوح الكاميرا
5. أعد التحميل وحاول مرة أخرى
```

### المشكلة: "Motion not detected"
```
❌ الخطأ: الحركات لا تُكتشف

✅ الحلول:
1. تحرك ببطء (حرّك رأسك بسلاسة)
2. عظّم الحركة (لفّ الرأس أكثر)
3. تأكد من وضع الوجه على الشاشة
4. جرب حركة أخرى أولاً
5. أعد المحاولة بحرارة أقل
```

### المشكلة: "Camera Permission Denied"
```
❌ الخطأ: الكاميرا مرفوضة

✅ الحلول:
1. أعد تحميل الصفحة
2. السماح للموقع بالوصول للكاميرا
3. تحقق من إعدادات الخصوصية
4. جرب متصفح آخر
5. أعد تشغيل الكمبيوتر
```
