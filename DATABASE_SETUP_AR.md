# 📦 دليل إعداد قاعدة بيانات SyrLink

## ✅ الحالة: تم إعداد قاعدة البيانات بنجاح!

### ما تم إنجازه
تم تهيئة قاعدة بيانات MongoDB بالكامل مع **11 مجموعة** (collections) وجميع الفهارس اللازمة:

| المجموعة | الغرض | عدد المستندات |
|---------|-------|---------------|
| `users` | ملفات المستخدمين والمصادقة | 13 |
| `posts` | منشورات المستخدمين | 6 |
| `connections` | طلبات الاتصال | 3 |
| `messages` | الرسائل المباشرة | 11 |
| `conversations` | محادثات الرسائل | 5 |
| `notifications` | إخطارات المستخدمين | 11 |
| `jobs` | الوظائف المتاحة | 8 |
| `companies` | ملفات الشركات | 0 |
| `recommendations` | التوصيات (كـ LinkedIn) | 0 |
| `endorsements` | التوصيات بالمهارات | 0 |
| `verification_requests` | طلبات التحقق من الحسابات | 0 |

---

## ✅ تأكيد حفظ البيانات

**نعم - جميع البيانات يتم حفظها في MongoDB وتبقى محفوظة عند:**
- ✓ تحديث الصفحة
- ✓ إعادة تشغيل السيرفر
- ✓ إغلاق المتصفح
- ✓ جلسات المستخدمين المختلفة

---

## 🔍 كيف تتحقق من حفظ البيانات؟

### اختبار بسيط من الواجهة الأمامية:
1. اذهب إلى http://localhost:3000
2. قم بتسجيل الدخول
3. أنشئ منشور جديد
4. **أعد تحميل الصفحة** (Refresh)
5. المنشور سيظل موجوداً ✓

هذا يثبت أن البيانات محفوظة في قاعدة البيانات وليست محفوظة محلياً في المتصفح فقط.

---

## 🔐 واجهات برمجية لإدارة قاعدة البيانات (API)

### 1. تهيئة قاعدة البيانات
```bash
curl -X POST http://localhost:8000/api/db/init \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json"
```

### 2. التحقق من حالة قاعدة البيانات
```bash
curl http://localhost:8000/api/db/status \
  -H "Authorization: Bearer TOKEN"
```

يعرض: جميع المجموعات وعدد المستندات والفهارس

### 3. حذف جميع البيانات (خطير ⚠️)
```bash
curl -X POST http://localhost:8000/api/db/clear-all \
  -H "Authorization: Bearer TOKEN"
```

## 👤 بيانات المسؤول (Admin)

### حساب المسؤول الافتراضي:
- **البريد الإلكتروني**: `admin@syrlink.com` (أو انظر `.env`)
- **كلمة المرور**: تحقق من ملف `.env`

### الحصول على رمز المسؤول:
```bash
curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "admin@syrlink.com",
    "password": "PASSWORD"
  }'
```

---

## 🗄️ تفاصيل الاتصال بقاعدة البيانات

### المتطلبات في `.env`:
```env
MONGO_URL=mongodb://username:password@localhost:27017/
DB_NAME=syrlink_db
```

### المجموعات التي تم إنشاؤها مع:
✓ حماية من التكرار (Unique constraints) على رسائل البريد الإلكتروني وبطاقات المستخدمين
✓ فهارس لاستعلامات سريعة
✓ فهارس للبحث النصي
✓ فهارس التاريخ للترتيب

---

## ✅ ماذا يتم حفظه في قاعدة البيانات

### ✓ يتم حفظه تلقائياً:
- ملفات المستخدمين والإعدادات
- المنشورات والمحتوى
- التعليقات والتفاعلات
- طلبات الاتصال
- الرسائل والمحادثات
- الإخطارات
- الوظائف المتاحة
- المعلومات
- التوصيات
- طلبات التحقق

### ✗ لا يتم حفظه (ذاكرة تخزين المتصفح فقط):
- رموز المصادقة (Tokens)
- حالة الواجهة المؤقتة
- مسودات النماذج

## 🚀 إعادة تعيين قاعدة البيانات (اختياري)

إذا أردت حذف جميع البيانات والبدء من جديد:

```bash
# الطريقة 1: عبر Python
python init_db.py

# الطريقة 2: عبر API
POST /api/db/clear-all
POST /api/db/init
```
